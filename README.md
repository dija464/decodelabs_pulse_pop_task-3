# Pulse — Interactive Web Elements

DecodeLabs Frontend Development Internship — Project 3

## Live Demo
https://pulse-pop.netlify.app

## Goal
Add interactivity to a webpage using JavaScript.

## Requirements Covered
- Buttons / toggles — dark mode toggle, FAQ accordion buttons
- Basic user interaction — click events on toggle, FAQ, navigation
- Dynamic content update — theme changes instantly, stats count up, FAQ expands/collapses

## What This Page Includes
- A dark/light mode toggle that remembers your choice using localStorage
- An FAQ section where clicking a question expands the answer
- A stats section where numbers count up when you scroll to them
- A mobile menu (hamburger) that opens and closes

## Built With
- HTML
- CSS (Grid, Flexbox, variables)
- JavaScript (DOM manipulation, event listeners)
